How To Run:

Locally:
- make sure both devices are on the same wifi
- generate a cert and key to replace in the code so we can use https
- run test_main.py
- go to the website test_main.py returns in the oculus browser
- find your local ip by running "ipconfig" in terminal
- input the wss://<local-ip>:8012
- reconnect and hit the button "Virtual reality"
- you will then be able to see the coord frames of the joints in your hands
    and get the matrix outputs in the terminal
